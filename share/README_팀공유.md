## 📦 이 폴더 안에 뭐가 있나

| 파일 | 용도 | 사용법 |
|---|---|---|
| `Mandata-AltData.webloc` | **macOS 바로가기** | 더블클릭 → 기본 브라우저에서 앱 열림 |
| `Mandata-AltData.url` | **Windows 바로가기** | 더블클릭 → 기본 브라우저에서 앱 열림 |
| `Mandata-AltData-Share.html` | **공유 카드 (QR + 안내)** | 더블클릭 → 예쁜 카드 + QR 코드 표시 |

## 🚀 어떻게 공유하지?

### 옵션 A — 슬랙/메신저로
URL 그대로 붙여넣기:
```
https://mandata-altdata.streamlit.app
```

### 옵션 B — 바로가기 파일 첨부
팀원의 OS에 맞춰 `.webloc`(Mac) 또는 `.url`(Windows) 파일을 첨부.
받는 사람은 더블클릭만 하면 됨 → 데스크톱에 두면 다음부터는 더 빠르게 접속.

### 옵션 C — 회의실 / 발표 화면
`Mandata-AltData-Share.html`을 더블클릭 → 큰 화면에 띄움 →
- QR 코드 보임 (모바일로 스캔)
- "앱 열기" 버튼 클릭 가능
- URL 복사 버튼

### 옵션 D — 브라우저 북마크 추천
팀원에게 "북마크바에 추가하세요"라고 안내하면 가장 빠른 접근.

## 👥 새 팀원을 위한 빠른 가이드

1. 위 URL 클릭 → 브라우저에 앱이 뜸 (Streamlit Cloud)
2. **Step 1** — 엑셀/CSV 업로드 (최대 5GB)
3. **Step 2** — Schema Intelligence가 자동으로 컬럼 추론
   - 회사명, 카테고리, 날짜, 매출 등을 자동 인식
4. **Step 3** — 데이터 품질 검증
5. **Step 4** — 분석 옵션 선택 (Market Signal, Earnings, Factor Research 등)
6. **Step 5** — 결과 확인 (차트 + 인사이트)
7. **Step 6** — Final Report 다운로드 (PPT / PDF, 한국어/영어)
